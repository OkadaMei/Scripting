# GiriGiri 动漫浏览器脚本文档

## 当前定位

当前版本：`2.7.0`。

本脚本是面向 **GiriGiri 爱动漫 / ani.girigirilove.com** 的 Scripting 单站点动漫浏览器。当前实现以公开网页浏览、系统播放与逐集离线管理为主，支持首页分区、分类浏览、搜索建议、番剧详情、标签搜索、字幕剧集列表、收藏、观看历史、直链视频与无 DRM HLS 下载。

> 边界：本脚本只处理公开页面及用户有权离线使用的可识别媒体来源；支持直链与无 DRM HLS 离线保存，绝不绕过验证码、访问控制或 DRM。

---

## 已适配能力

### 在线浏览

- `page/library/index.tsx` 作为首页浏览页，加载 `ani.girigirilove.com` 公开首页并按分区展示番剧。
- `class/hanime.ts` 保留旧导出名以降低改动范围，但内部已切换为 GiriGiri / MacCMS 网页适配层。
- 首页解析公开卡片、横幅、统计卡、刷新按钮和官网打开入口。
- 遇到验证码 / 403 / 429 等访问限制时，只提示用户用 Safari 打开官网确认可访问后再重试。

### 搜索、分类与标签

- `page/search/index.tsx` 支持搜索框回车提交、搜索历史、分类浏览、排序浏览和分页加载。
- 关键词搜索使用公开建议接口：`/index.php/ajax/suggest?mid=1&wd=...&limit=30`。
- 清空关键词后使用 MacCMS 分类页 `/show/...` 浏览公开列表。
- 分类选项覆盖日番、美番、剧场版、真人番剧、BD 副音轨、演唱会 / 周边。
- 排序支持默认、最新、最热和评分。
- 详情页标签使用三列虚线网格，点击单个标签只进入对应标签结果页。

### 详情、系统播放与离线下载

- `page/hanime/video_detail.tsx` 展示标题、完整标题复制、封面、简介、标签、字幕剧集入口和相关推荐。
- 番剧详情路由：`/GV{id}/`；剧集播放路由：`/playGV{id}-{sid}-{nid}/`。
- 在线剧集先解析可播放的 m3u8 / mp4 直链，再交给 iOS 系统播放链路；打开播放入口会写入本地观看历史。
- 下载按“番剧 ID + 字幕/线路 + 集数”独立管理，不同剧集不会覆盖。
- 直链视频保存为本地文件；HLS 先保存本地清单、分片、初始化段及允许的 AES-128 key，再优先以 ffmpeg `-c copy` 无损封装 MP4；无法封装时保留 HLS 离线播放包。
- 不处理 DRM、Sample-AES / FairPlay、访问控制规避或受保护资源。

### 本地记录

- `class/hanime_database.ts` 负责收藏、观看历史、搜索历史与逐集下载记录。
- `page/saved/index.tsx` 和下载 Tab 聚合本地收藏、历史、MP4 文件与 HLS 离线播放包；可播放、导出或删除。
- 本地数据目录使用 `GiriGiri` 命名空间。

### 设置与声明

- `page/setting/index.tsx` 提供官网入口、危险区域、本地数据维护、功能边界、免责声明和版本信息。
- 版本号来自 `script.json` 的 `version` 字段，当前显示 `2.7.0`。
- 首次进入浏览页会显示站点访问提示；可在设置页重置。

## 功能边界

- 已实现无需登录的公开页面链路、本机收藏、搜索历史、观看历史、逐集下载与离线播放管理。
- 暂不支持登录账号、评论同步、云端收藏同步、弹幕互动、订阅追番同步、后台批量任务、跨会话断点续传和受保护内容保存。
- 离线下载仅适用于可识别的直链视频、无 DRM HLS 与符合站点规则的 AES-128 清单；MP4 封装受来源、ffmpeg 与系统导出兼容性影响，必要时保留 HLS 离线播放包。
- 遇到验证码或站点访问限制时，需要先在 Safari 打开官网完成验证或确认访问正常；脚本不提供绕过访问控制、破解或再分发能力。

## 代码规范

- 文件名使用 `snake_case`。
- 变量和函数使用 `camelCase`。
- 类和类型使用 `PascalCase`。
- 两空格缩进，优先使用明确类型和现有共享组件。
- UI 页面优先使用 `List listStyle="inset"`、`Section`、摘要卡、空/错/加载状态组件。
- `List` 中并排按钮需使用 `buttonStyle="borderless"` 或 `plain`，避免串触发。
- 标签与剧集网格避免堆多个默认 `NavigationLink`。
- 站点抓取集中在 `class/hanime.ts`；页面不直接散落复杂 HTML 解析。
- 当前保留 `hanime*` 类型 / 文件名是低风险兼容旧页面的过渡方案，不代表当前站点语义。

  4. 详情页能解析标题、封面、简介与剧集。

  6. 收藏、观看历史和搜索历史可写入与展示。
  7. 设置页功能边界明确说明不下载、不破解、不绕过验证。
