## 🚀 使用

1. 在 App Store 下载并安装 **Scripting**。
2. 在上方 **Scripts** 列表中选择需要的脚本。
3. 点击 **📥 安装**。
4. 浏览器将自动跳转至 **Scripting** 并完成导入。
5. 如脚本需要 API Key 或其他配置，请参考脚本内说明。
