Scripting Scripts

适用于 Scripting 的脚本合集。

Scripting 是一款运行于 iPhone 和 iPad 上的 JavaScript / TypeScript 开发平台，可以编写脚本、创建 Widget、构建快捷指令扩展，并调用丰富的 iOS 原生能力。

获取 Scripting

App Store：

https://apps.apple.com/tr/app/scripting/id6479691128

关于本仓库

本仓库用于分享本人编写的 Scripting 脚本。

所有脚本均按功能分类存放，每个目录对应一个独立项目，可直接导入 Scripting 使用。

仓库会持续维护，并不断添加新的脚本与示例。

仓库结构

.
├── Script A
├── Script B
├── Script C
└── ...

每个目录通常包含：

* 脚本源码
* 项目配置
* 相关资源（如有）

使用方式

1. 安装 Scripting。
2. 下载或克隆本仓库。
3. 将需要的脚本导入 Scripting。
4. 根据脚本内说明完成必要配置（如 API Key、权限等）。
5. 运行脚本。

说明

部分脚本可能依赖：

* 网络连接
* 定位权限
* 相册权限
* 第三方 API 或服务

具体要求请以对应脚本目录中的说明为准。
